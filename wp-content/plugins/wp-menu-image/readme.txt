This fabulous plugin has been developed by <a href="https://www.yudiz.com/">Yudiz Solutions Ltd.</a> WP Menu Image plugin helps you to add image to your menu. 

How to use this plugin?
Follow the below <b>steps</b>:
1) Install and activate the plugin.
